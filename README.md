# DESIGN AND SIMULATION OF A HIERARCHICAL CAMPUS NETWORK USING VLANS, VTP, DHCP, AND INTER-VLAN ROUTING

## 📌 Project Overview

This project demonstrates the design and simulation of a hierarchical campus network using Cisco Packet Tracer. The network is structured using core, distribution, and access layers to ensure scalability, performance, and efficient management.

## 🛠️ Technologies Used

* Cisco Packet Tracer (.pkt file)
* VLAN (Virtual Local Area Network)
* VTP (VLAN Trunking Protocol)
* DHCP (Dynamic Host Configuration Protocol)
* Inter-VLAN Routing

## 🧱 Network Design

The network follows a hierarchical model:

* **Core Layer** → High-speed backbone
* **Distribution Layer** → Routing and policy control
* **Access Layer** → End device connectivity

## ⚙️ Features Implemented

* VLAN segmentation for different departments
* VTP for automatic VLAN management
* DHCP for automatic IP address assignment
* Inter-VLAN routing for communication between VLANs
* Trunking between switches

## 📂 Files Included

* `campus_network.pkt` → Main simulation file

## ▶️ How to Run

1. Install Cisco Packet Tracer
2. Open the `.pkt` file
3. Run the simulation mode to observe network behavior

## 🎯 Learning Outcomes

* Understanding hierarchical network design
* Configuring VLANs and trunking
* Implementing VTP and DHCP
* Performing Inter-VLAN routing
